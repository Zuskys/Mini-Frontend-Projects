## Mini Frontend Projects

A collection of small frontend applications built to practice core JavaScript concepts and UI development.
This repository is not a single project. Each folder represents an isolated application. Most projects are built with vanilla JavaScript to strengthen fundamentals, but some may use frameworks.

## What you may find here

- Vanilla JavaScript applications
- UI interaction patterns
- State management approaches
- Browser APIs experimentation

## Why this repository exists

A single place to strengthen fundamentals by building many small applications.
